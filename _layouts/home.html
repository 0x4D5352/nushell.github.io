Installation Overview

Pre-built binaries

Please Note: Nu works on Windows 10 and does not currently have Windows 7/8.1 support.

Download the current released .zip-file from the release page and extract it, for example, to:

C:\Program Files

And then add the folder of nu to your PATH. Once we have done that, we can run Nu using the nu command:

> nu
  C:\Users\user>

Building Nu locally

You can of course build Nu locally. To do this please look over here

Homebrew

If you use Homebrew on macOS or Linux, you can install the binary by running 

> brew install nushell

Further information

For a quick start we recommend you to read our book. It contains all the information you need for setting up Nushell and start working with it.

If you've already gone through Nu Book and you still want to have more examples of what Nu is capable off, you can take a look in the cookbook.

We have active discussions on Discord and Twitter, where you can get help from developers and other Nu users if you run into problems. Furthermore you can bring in your own ideas and visions for evolving Nu and work with us together on building this modern shell.
